# Project Overview

Tessellum is a desktop application focused on personal knowledge management. Its main purpose is to help users create, organize, and consult notes in a practical and understandable way. Instead of treating notes as isolated files, the application allows users to connect ideas, search information quickly, and move through their content more easily. In general terms, the project is intended to solve a common problem: having useful information spread across many documents without a simple way to organize it and find it again.

The project is a local-first desktop application. This means that it runs on the user's own computer and keeps the information in regular folders and Markdown files chosen by the user. In this way, Tessellum aims to offer the convenience of a modern note-taking and knowledge management tool while allowing users to keep control of their own data. From the user's point of view, the application makes it possible to write and edit notes, browse folders, search content, view the relationships between notes, and adapt the interface to different work preferences.

According to the current state of the project, Tessellum is not presented as software made for a specific external client. It should instead be understood as an independent project with a clear practical purpose and with the potential to be used in real environments in the future. The repository shows that it has been prepared as a complete application rather than as a simple academic prototype, since it includes release and installation preparation, material for usability evaluation, and features designed for real day-to-day use.

Among the most relevant aspects of the project are its focus on usability, customization, and future growth. The application includes a guided demo note for new users, supports different ways of exploring information, and has been prepared for testing with users through a dedicated usability questionnaire. At the time of writing, the available project material does not indicate awards, monetization, or development for a real client, but it does show a clear intention for the application to be mature, distributable, and useful beyond the classroom context. Overall, Tessellum can be understood as a desktop tool created to help people manage knowledge in a more organized, visual, and accessible way.
